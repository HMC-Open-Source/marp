# How to learn marp with this repo
- Download the Marp VS Code extention
    - Close and reopen VS Code after download
- Fork this repo to your GitHub account
- Clone your fork to your laptop
    - You'll have to choose a spot for it!
- Open the folder in VS Code
- View the ``demo.md`` folder
- Ctrl/Cmd + Shift + V to open up a preview of the .md file as slides

Review the ``demo.md`` file alongside the preview to learn how to use marp.

Feel free to make edits and submit PRs! Except, I'm not sure if I'll be good
about reviewing them.
